# Placement Tracker

A platform for tracking placement preparation progress.

Features:

- Authentication
- Role Management
- DSA Tracking
- DBMS Tracking
- OS Tracking
- CN Tracking
- OOP Tracking
- Screenshot Verification
- Approval Workflow
- Streak Tracking
- Heatmaps
- Badges
- Notices

Tech Stack:

- React
- Node.js
- Express
- PostgreSQL
- JWT
- Cloudinary
